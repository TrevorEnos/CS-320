How can I ensure that my code, program, or software is functional and secure?
  One of the best ways is through JUnit testing. This ensures that any limits that you have set are working correctly when given input. It is also just general good practice to make sure that the terminal is clear of errors before running. If any bugs are left it could create vulnerabilities for        users.

How do I interpret user needs and incorporate them into a program?
  Determing users needs begins with communication. We need to know what the goal of the user is or what is the purpose of the app. From here we can build a guide to how we implement/design the code.

How do I approach designing software?
  Designing software begins at figuring out our needs. Then we can deside the overall UI and functionality. This can be done through hosting a meeting with a client about the app. Or for a personal project we would need to create a list of things that are needed for our goal functionality.
